# Making users scroll
- break the fold(initially visible section) - breaking means content placed below fold partially visible in fold
- show scroll cues like a floating button, horizontal scrollbar etc
  - scroll cues less effective than breaking fold
- Use banner images in blogs which take up most of the fold so users scroll (like newstack.io)
- show in-page navigation (Increases Easy Scan) / sticky CTA (Accessible action when user is ready)
